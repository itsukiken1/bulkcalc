# BulkCalc - Batch Profit Calculator for Cross-Border Sellers

**[Use it free → itsukiken1.github.io/bulkcalc](https://itsukiken1.github.io/bulkcalc/)**

Calculate profit margins for multiple products at once when sourcing from China (1688/Alibaba).

## Features

- Paste or upload a product list (CSV/TXT)
- Batch calculate: sourcing cost → shipping → customs → platform fees → profit
- **7 markets**: USA, UK, Germany, Japan, Australia, UAE, Saudi Arabia
- **6 platforms**: Amazon FBA, Amazon FBM, Shopify, Temu, eBay, TikTok Shop
- Sea freight & air freight estimates
- Real-time CNY/USD exchange rates
- Sort, filter, export to CSV, copy to clipboard
- Summary stats: profitable/unprofitable count, avg margin, best product

## Why?

Cross-border sellers evaluate dozens of products before choosing what to source. Doing profit calculations one-by-one in a spreadsheet is slow. BulkCalc lets you paste your entire product list and see all margins instantly.

## Privacy

- No data collection
- No signup required
- All calculations run locally in your browser
- Exchange rates fetched from open API (no personal data sent)
